# Manso Electricista

Landing page para servicios eléctricos en Mendoza. Diseño moderno, foco en confianza, contacto rápido y SEO local.

## Stack

- Angular 21
- CSS plano (sin frameworks)
- Deploy en Netlify

## URLs

- Producción: https://manso-electricista.netlify.app
- Repo: https://github.com/alfredo-vazquez-25/manso-electricista

## Desarrollo local

```bash
npm install
npm start
```

Luego abrir `http://localhost:4200/`.

## Build

```bash
npm run build
```

Salida en `dist/julian-electricista/browser`.

## Deploy automático

Netlify está conectado al repo. Cada push a `main` genera un deploy automático.

## Notas

- Imágenes en `public/assets/images/`.
- Config de Netlify en `netlify.toml`.
